# Release Notes

## v1.1.0 (April 2026)

### ✨ Minor Release Highlights
- Added **Change Password** feature for all authenticated roles:
  - User
  - Trainer
  - Admin
- Added secure password update flow:
  - Verifies current password
  - Confirms new password match
  - Enforces minimum length (6 chars)
- Added role-safe redirect to the correct dashboard after password update.
- Added dashboard links to access password change screen quickly.

### ✅ Validation behavior
- Rejects empty fields
- Rejects incorrect current password
- Rejects mismatched new/confirm password

### 🛠 Tech
- Flask + Flask-Login + Werkzeug password hashing (`pbkdf2:sha256`)
