# Release Notes

## v1.0.1 (April 2026)

### ✅ What’s included
- Implemented complete role-based flow:
  - **User**: registration, login, and user dashboard
  - **Admin**: trainer management and trainer assignment to users
  - **Trainer**: login and view assigned users
- Added/updated trainer administration:
  - Create trainer accounts
  - Delete trainer accounts
  - Assign trainer to user
- Added/confirmed dashboard routing by role:
  - `/dashboard` redirects correctly for admin/trainer/user
- Added member creation endpoint:
  - `POST /members` (accepts JSON or form data)
- Existing APIs available:
  - `GET /members`
  - `GET /trainers`
  - `GET /health`
- App initializes DB and creates default admin account if missing.

### 🛠 Notes
- Built with Flask + SQLAlchemy + Flask-Login.
- SQLite database is created under the Flask instance path.
