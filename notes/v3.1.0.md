# Release Notes

## v3.1.0 (April 2026)

### Minor Features
- Added **Edit Workout Plan** support for trainers.
- Trainers can now update:
  - Workout plan title
  - Workout plan details
  - Assigned user (only from trainer’s assigned users)

### Security/Validation
- Trainers can edit only their own workout plans.
- User reassignment is restricted to users assigned to the logged-in trainer.

### Notes
- This is a minor release.
- Existing create/delete workout plan flows remain unchanged.
