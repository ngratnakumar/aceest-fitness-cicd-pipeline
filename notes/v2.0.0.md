# Release Notes

## v2.0.0 (April 2026)

### Major Features
- Added **Workout Plan** management for trainers.
- Trainers can now:
  - Create multiple workout plans
  - Assign each workout plan to a specific user
  - View all workout plans they created
  - Delete their own workout plans
- Users can now see their assigned workout plans on the user dashboard.

### Dashboard Updates
- User dashboard shows workout plans assigned to the logged-in user.
- Trainer dashboard includes workout plan creation and assignment form.
- Trainer dashboard lists all created workout plans.

### Notes
- This is a major release.
- Existing login, registration, password change, and admin trainer assignment features remain available.
